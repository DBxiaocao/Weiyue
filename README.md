微阅是一款集新闻，视频和妹子的阅读类软件，基于 Material Design + RxJava2.x + Retrofit2.x + Glide＋MVP，基本涵盖了当前 Android 端开发最常用的主流框架。

### 目前模块包括：
文章，视频，美图，可本地收藏分享
